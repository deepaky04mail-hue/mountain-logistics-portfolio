# Dashboard Description

The dashboard is divided into two interactive pages.

## Executive Overview

This page provides a high-level summary of overall business performance.

It includes:

- Revenue
- Profit
- Profit Margin
- Shipment Count
- Customer Satisfaction
- Profit by Region
- Delivery Status Distribution
- Top Companies by Profit

Interactive filters allow users to explore results by region, company, weather condition, and logistics upgrade.

## Operational Analysis

This page focuses on operational performance.

Visualizations include:

- Weather Impact Analysis
- Logistics Upgrade Comparison
- Revenue Category Distribution
- Delay Category Distribution
- Risk vs Profit Analysis
- Profit Category Distribution

These charts help identify operational patterns that influence profitability and delivery performance.
