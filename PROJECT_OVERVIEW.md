# Project Overview

The Mountain Logistics Analytics Dashboard is designed to monitor operational performance for logistics services operating across different mountain regions.

The project combines SQL-based data analysis with interactive dashboarding in Google Looker Studio. Instead of reviewing raw shipment records, users can quickly explore revenue, profit, customer satisfaction, delays, and regional performance through visual reports.

The dashboard includes interactive filters, allowing managers to analyze results by mountain region, company, weather conditions, and logistics upgrades.

The main objective of this project is to demonstrate practical SQL querying, business analysis, and dashboard design using real business scenarios.
